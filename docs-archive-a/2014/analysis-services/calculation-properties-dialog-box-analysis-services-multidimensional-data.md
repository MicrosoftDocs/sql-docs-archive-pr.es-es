# <a name="calculation-properties-dialog-box-analysis-services---multidimensional-data"></a>Cuadro de diálogo Propiedades de cálculo (Analysis Services - Datos multidimensionales)
  Utilice el cuadro de diálogo **Propiedades de cálculo** de [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] para ver o modificar las propiedades de las carpetas para mostrar y los grupos de medida asociados de los cálculos definidos en la pestaña **Cálculos** del Diseñador de cubos para el cubo seleccionado. Para abrir el cuadro de diálogo **Propiedades de cálculo** , haga clic en **Propiedades de cálculo** en el panel **Barra de herramientas** de la pestaña **Cálculos** del Diseñador de cubos.  
  
## <a name="options"></a>Opciones  
 **Nombre de cálculo**  
 Muestra el nombre del miembro calculado o del conjunto con nombre.  
  
> [!NOTE]  
>  Los comandos del script no se muestran.  
  
 **Carpeta para mostrar**  
 Escriba el nombre de la carpeta en la que debe mostrarse el cálculo en las aplicaciones cliente, incluyendo [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] y [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)].  
  
 **Grupo de medida asociado**  
 Seleccione el grupo de medida en el que el miembro calculado o el conjunto con nombre deberá mostrarse en las aplicaciones cliente, entre las que se incluyen [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] y [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)].
