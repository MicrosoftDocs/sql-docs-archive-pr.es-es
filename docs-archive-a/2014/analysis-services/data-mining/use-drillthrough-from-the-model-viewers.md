# <a name="use-drillthrough-from-the-model-viewers"></a><span data-ttu-id="9a8a9-102">Usar la obtención de detalles desde los visores de modelos</span><span class="sxs-lookup"><span data-stu-id="9a8a9-102">Use Drillthrough from the Model Viewers</span></span>
  <span data-ttu-id="9a8a9-103">Dependiendo del tipo de modelo, es posible utilizar la obtención de detalles desde los visores de exploración de la pestaña **Visor de modelos de minería de datos** del Diseñador de minería de datos para explorar los casos usados en el modelo de minería de datos o para ver columnas adicionales en la estructura de minería de datos.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-103">Depending on the model type, you can use drillthrough from the browse viewers on the **Mining Model Viewer** tab of Data Mining Designer to explore the cases used in the mining model or to see additional columns in the mining structure.</span></span> <span data-ttu-id="9a8a9-104">Aunque muchos tipos de modelos no admiten la obtención de detalles debido a que los patrones del modelo no se pueden vincular directamente con casos concretos, los tipos de modelos siguientes sí lo hacen.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-104">Although many model types do not support drillthrough because the patterns in the model cannot be directly linked to specific cases, the following model types do support drillthrough.</span></span>  
  
 <span data-ttu-id="9a8a9-105">Tenga en cuenta que se debe haber habilitado la obtención de detalles en el modelo, y que debe disponer de los permisos apropiados.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-105">Note that drillthrough must have been enabled on the model, and you must have the appropriate permissions.</span></span> <span data-ttu-id="9a8a9-106">Es posible que la opción para obtener detalles también esté deshabilitada si el modelo se encuentra en un estado no procesado, independientemente de si el modelo se procesó previamente y tiene contenido.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-106">The drillthrough option might also be disabled if the model is in an unprocessed state, regardless of whether the model was previously processed and has content.</span></span> <span data-ttu-id="9a8a9-107">Para recuperar los datos de los casos del modelo mediante la obtención de detalles, la memoria caché de la estructura y del modelo debe estar actualizada.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-107">To retrieve model case data by using drillthrough, the cache of the structure and model must be current.</span></span>  
  
### <a name="use-drillthrough-in-the-microsoft-tree-viewer"></a><span data-ttu-id="9a8a9-108">Utilizar la obtención de detalles en el Visor de árboles de Microsoft</span><span class="sxs-lookup"><span data-stu-id="9a8a9-108">Use drillthrough in the Microsoft Tree Viewer</span></span>  
  
1.  <span data-ttu-id="9a8a9-109">En el Diseñador de minería de datos, seleccione un modelo de árboles de decisión y, a continuación, seleccione **Examinar modelo** para abrir el modelo en el **Visor de árboles de Microsoft**.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-109">In Data Mining Designer, select a decision trees model, and select **Browse Model** to open the model in the **Microsoft Tree Viewer**.</span></span> <span data-ttu-id="9a8a9-110">En SQL Server Management Studio, haga clic con el botón derecho en el modelo y seleccione **Examinar**.</span><span class="sxs-lookup"><span data-stu-id="9a8a9-110">In SQL Server Management Studio, right-click the model, and select **Browse**</span></span>
