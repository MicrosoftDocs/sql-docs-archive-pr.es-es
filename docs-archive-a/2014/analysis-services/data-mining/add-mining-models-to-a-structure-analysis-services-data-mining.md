 <span data-ttu-id="ae1d3-128">Al agregar nuevos modelos de minería de datos a una estructura de minería de datos existente, debe especificar cada columna de datos que debe utilizar el modelo.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-128">When you add new mining models to an existing mining structure, you must specify how each column of data should be used by the model.</span></span> <span data-ttu-id="ae1d3-129">Según el tipo de algoritmo que se elija para el modelo, algunas de estas opciones se pueden efectuar de forma predeterminada.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-129">Depending on the type of algorithm you choose for the model, some of these choices may be made by default.</span></span> <span data-ttu-id="ae1d3-130">Si no define un tipo de uso para una columna, esta no se incluirá en la estructura de minería de datos.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-130">If you do not specify a usage type for a column, the column will not be included in the mining structure.</span></span> <span data-ttu-id="ae1d3-131">Sin embargo, los datos de la columna pueden seguir disponibles para la obtención de detalles, si el modelo lo admite.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-131">However, the data in the column can still be available for drillthrough, if the model supports it.</span></span>

-   <span data-ttu-id="ae1d3-133">Las columnas de clave contienen un único identificador para cada fila de una tabla.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-133">Key columns contain a unique identifier for each row in a table.</span></span> <span data-ttu-id="ae1d3-134">Algunos modelos de minería de datos, como los que se basan en la agrupación en clústeres de secuencia o en algoritmos de serie temporal, pueden contener varias columnas de clave.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-134">Some mining models, such as those based on the sequence clustering or time series algorithms, can contain multiple key columns.</span></span> <span data-ttu-id="ae1d3-135">Sin embargo, estas claves no son claves compuestas en el sentido relacional, pero deben seleccionarse de esta forma para proporcionar soporte a las series temporales y al análisis de agrupación en clústeres de secuencia.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-135">However, these multiple keys are not compound keys in the relational sense, but instead must be selected so as to provide support for time series and sequence clustering analysis.</span></span>

     <span data-ttu-id="ae1d3-140">Esta característica resulta especialmente práctica cuando los conjuntos de datos contienen más columnas de las que realmente se necesitan para generar un modelo de minería de datos.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-140">This feature is particularly handy when datasets contain more columns than you really need to build a mining model.</span></span> <span data-ttu-id="ae1d3-141">La característica **Sugerir** calcula una puntuación numérica, de 0 a 1, que describe la relación de cada columna del conjunto de datos con la columna de predicción.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-141">The **Suggest** feature calculates a numeric score, from 0 to 1, that describes the relationship between each column in the dataset and the predictable column.</span></span> <span data-ttu-id="ae1d3-142">Basándose en esta puntuación, la característica sugiere las columnas que se pueden usar como entrada para el modelo de minería de datos.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-142">Based on this score, the feature suggests columns to use as input for the mining model.</span></span> <span data-ttu-id="ae1d3-143">Si utiliza la característica **Sugerir** , puede usar las columnas sugeridas, modificar las selecciones para que se ajusten a sus necesidades o pasar por alto las sugerencias.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-143">If you use the **Suggest** feature, you can use the suggested columns, modify the selections to fit your needs, or ignore the suggestions.</span></span>  
  
-   <span data-ttu-id="ae1d3-144">Las columnas de predicción contienen la información que se intenta predecir en el modelo de minería de datos.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-144">Predictable columns contain the information that you try to predict in the mining model.</span></span> <span data-ttu-id="ae1d3-145">Puede seleccionar varias columnas con los atributos de predicción.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-145">You can select multiple columns as the predictable attributes.</span></span> <span data-ttu-id="ae1d3-146">Los modelos de clústeres son la excepción, en el sentido de que un atributo de predicción es opcional.</span><span class="sxs-lookup"><span data-stu-id="ae1d3-146">Clustering models are the exception in that a predictable attribute is optional.</span></span>  
  
     <span data-ttu-id="ae1d3-147">Según el tipo de modelo, podría ser necesario que la columna de predicción fuera un tipo de datos específico: por ejemplo, un modelo de regresión lineal requiere una columna numérica como valor de predicción; el algoritmo Bayes Naïve requiere un valor discreto (y todas las entradas también deben ser discretas).</span><span class="sxs-lookup"><span data-stu-id="ae1d3-147">Depending on the model type, the predictable column might need to be a specific data type: for example, a linear regression model requires a numeric column as the predicted value; Naïve Bayes algorithm requires a discrete value (and all the inputs must be discrete as well).</span></span>
