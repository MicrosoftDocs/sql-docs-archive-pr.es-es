# <a name="sql-server-management-studio-web-browser"></a>Explorador web de SQL Server Management Studio
  [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] contiene una versión de [!INCLUDE[msCoName](../includes/msconame-md.md)] Internet Explorer. Este explorador web permite ir a direcciones URL y ver los temas de ayuda de MSDN Library sin salir de [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)]. Para obtener acceso al explorador web, seleccione **Explorador web** en el menú **Ver** y, a continuación, haga clic en **Mostrar el explorador**.
