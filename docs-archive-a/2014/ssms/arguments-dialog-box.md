# <a name="arguments-dialog-box"></a>Argumentos, cuadro de diálogo
  El cuadro de diálogo **Argumentos** especifica argumentos nuevos o existentes de una herramienta. Este cuadro de diálogo aparece al iniciar una herramienta externa que tenga la opción **Solicitar argumentos** del cuadro de diálogo **Herramientas externas** activada.  
  
## <a name="options"></a>Opciones  
 **Argumentos**  
 Enumera las variables actuales especificadas para la herramienta en el cuadro de diálogo **Herramientas externas** . Para obtener una lista completa de los argumentos predefinidos y sus definiciones, vea [Arguments for External Tools](menu-help/external-tools.md). También puede especificar un argumento personalizado.  
  
 **Línea de comandos**  
 Muestra el valor de los argumentos especificados en el cuadro de diálogo **Argumentos** .
