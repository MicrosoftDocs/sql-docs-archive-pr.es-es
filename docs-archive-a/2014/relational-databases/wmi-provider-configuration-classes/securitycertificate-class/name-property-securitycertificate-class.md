# <a name="name-property-securitycertificate-class"></a>Propiedad Name (clase SecurityCertificate)
  Obtiene el nombre del certificado de seguridad.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.Name [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase SecurityCertificate](securitycertificate-class.md) que representa un certificado de seguridad.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el nombre del certificado de seguridad.
