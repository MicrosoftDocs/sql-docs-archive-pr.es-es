# <a name="properties-property-servernetworkprotocolipaddress-class"></a>Propiedad Properties (clase ServerNetworkProtocolIPAddress)
  Obtiene las propiedades asociadas a una dirección IP.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.Properties [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase ServerNetworkProtocolIPAdress](servernetworkprotocolipaddress-class.md) que representa una dirección IP del protocolo de red en una instancia de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Matriz de objetos que representan las propiedades admitidas por la dirección IP.
