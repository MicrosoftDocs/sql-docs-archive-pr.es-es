# <a name="annotation-interpretation-sqlxml-40"></a>Interpretación de anotaciones (SQLXML 4.0)
  Los temas de esta sección describen la forma en que la carga masiva XML interpreta las anotaciones del esquema XSD. El comportamiento que aquí se describe también se aplica a las anotaciones del esquema XDR.  
  
> [!NOTE]  
>  La información de estos temas describe únicamente las anotaciones que la carga masiva XML utiliza en su procesamiento. Para obtener una lista completa de anotaciones para el esquema XSD que son compatibles con SQLXML 4,0, vea [utilizar anotaciones en esquemas xsd &#40;SQLXML 4,0&#41;](../../sqlxml-annotated-xsd-schemas-using/using-annotations-in-xsd-schemas-sqlxml-4-0.md). Para obtener una lista de las anotaciones admitidas para los esquemas XDR, consulte [esquemas XDR anotados &#40;en desuso en SQLXML 4,0&#41;](../../sqlxml/annotated-xsd-schemas/annotated-xdr-schemas-deprecated-in-sqlxml-4-0.md).

 [SQL: Relationship y la regla de ordenación de claves &#40;SQLXML 4,0&#41;](annotation-interpretation-sql-relationship-and-key-ordering-rule.md)  
 Describe la forma en que se interpreta la anotación `sql:relationship` en la carga masiva XML.  
  
 [SQL: asignado &#40;SQLXML 4,0&#41;](annotation-interpretation-sql-mapped.md)  
 Describe la forma en que se interpreta la anotación `sql:mapped` en la carga masiva XML.  
  
 [SQL: Limit-Field y SQL: Limit-Value &#40;SQLXML 4,0&#41;](annotation-interpretation-sql-limit-field-and-sql-limit-value.md)  
 Describe la forma en que se interpretan las anotaciones `sql:limit-field` y `sql:limit-value` en la carga masiva XML.  
  
 [SQL: Overflow-Field &#40;SQLXML 4,0&#41;](annotation-interpretation-sql-overflow-field.md)  
 Describe la forma en que se interpreta la anotación `sql:overflow` en la carga masiva XML.  
  
 [Otras anotaciones &#40;SQLXML 4,0&#41;](annotation-interpretation-other-annotations.md)  
 Describe la forma en que se interpretan las siguientes anotaciones en la carga masiva XML: `sql:id-prefix`, `sql:use-cdata`, `sql:url-encode`, `sql:is-mapping-schema`, `sql:key-fields`.
