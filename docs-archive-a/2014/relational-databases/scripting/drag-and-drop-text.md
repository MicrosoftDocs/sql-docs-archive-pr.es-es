# <a name="drag-and-drop-text"></a>Arrastrar y colocar texto
  Se puede seleccionar texto para arrastrarlo y colocarlo en otra ubicación. Se puede arrastrar y colocar texto:  
  
-   Desde el Explorador de objetos al editor para crear una consulta.  
  
-   Desde una ubicación a otra diferente del editor actual.  
  
-   Entre editores.  
  
-   A la Papelera de reciclaje de [!INCLUDE[msCoName](../../includes/msconame-md.md)] Windows.  
  
### <a name="to-drag-and-drop-text"></a>Para arrastrar y colocar texto  
  
1.  Seleccione el texto que desea mover, con el mouse (ratón) o con el teclado.  
  
2.  Haga clic en el texto resaltado y mantenga presionado el botón del mouse.  
  
3.  Mueva el cursor hasta el lugar donde desea colocar el texto.  
  
4.  Suelte el botón del mouse para colocar el texto.  
  
 Al arrastrar texto se mueve, es decir, se borra de la ubicación anterior y se mueve a la nueva ubicación. Al presionar la tecla CTRL mientras se arrastra, el texto se copia a la nueva ubicación. Sin embargo, si arrastra un objeto desde el Explorador de objetos esta norma no se cumple, ya que el nombre del objeto no se elimina del Explorador de objetos.
