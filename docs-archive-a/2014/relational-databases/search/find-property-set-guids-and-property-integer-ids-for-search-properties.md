##  <a name="adding-a-property-to-a-search-property-list"></a><a name="examples"></a><span data-ttu-id="eb5d1-166">Agregar una propiedad a una lista de propiedades de búsqueda</span><span class="sxs-lookup"><span data-stu-id="eb5d1-166">Adding a Property to a Search Property List</span></span>

```  
ALTER SEARCH PROPERTY LIST PropertyList1   
  ADD 'Author'  
    WITH (  
          PROPERTY_SET_GUID = 'F29F85E0-4FF9-1068-AB91-08002B27B3D9',  
          PROPERTY_INT_ID = 4,

         )  
GO  
```
