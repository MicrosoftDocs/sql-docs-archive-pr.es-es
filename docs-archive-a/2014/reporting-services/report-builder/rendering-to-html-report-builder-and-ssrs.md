##  <a name="html-specific-rendering-rules"></a><a name="HTMLSpecificRenderingRules"></a><span data-ttu-id="bc311-129">Reglas de representación específicas de HTML</span><span class="sxs-lookup"><span data-stu-id="bc311-129">HTML-Specific Rendering Rules</span></span>  
 <span data-ttu-id="bc311-130">Durante la representación se aplican las siguientes reglas específicas de HTML:</span><span class="sxs-lookup"><span data-stu-id="bc311-130">The following HTML-specific rules are applied when rendering:</span></span>
