# <a name="text-box-properties-dialog-box-alignment-report-builder-and-ssrs"></a>Cuadro de diálogo Propiedades de cuadro de texto, Alineación (Generador de informes y SSRS)
  Seleccione **Alineación** en el cuadro de diálogo **Propiedades de cuadro de texto** para situar los valores dentro de la celda o región de datos seleccionada.  
  
## <a name="options"></a>Opciones  
 **Horizontal**  
 Especifique si el valor está colocado horizontalmente dentro de una celda o región de datos seleccionada. La opción seleccionada de forma predeterminada es **Predeterminada** . El valor de Predeterminada es **General**. Para obtener más información sobre cómo se calcula la alineación del texto general, vea [Aplicar formato a texto y a marcadores de posición &#40;Generador de informes y SSRS&#41;](report-design/formatting-text-and-placeholders-report-builder-and-ssrs.md). Haga clic en el botón **Expresión** (*fx*) para editar la expresión.  
  
 **Vertical**  
 Especifique si el valor está colocado verticalmente dentro de una celda o región de datos seleccionada. La opción seleccionada de forma predeterminada es **Predeterminada** . El valor de Predeterminada es **Superior**. Haga clic en el botón **Expresión** (*fx*) para editar la expresión.  
  
 **Left**  
 Especifique el relleno (o espacio) que hay entre el borde izquierdo de una celda y el valor mostrado dentro de la celda. El valor predeterminado es 2 puntos. Haga clic en el botón **Expresión** (*fx*) para editar la expresión.  
  
 **Right**  
 Especifique el relleno (o espacio) que hay entre el borde derecho de una celda y el valor mostrado dentro de la celda. El valor predeterminado es 2 puntos. Haga clic en el botón **Expresión** (*fx*) para editar la expresión.  
  
 **Top** (Principales)  
 Especifique el relleno (o espacio) que hay entre el borde superior de una celda y el valor mostrado dentro de la celda. El valor predeterminado es 2 puntos. Haga clic en el botón **Expresión** (*fx*) para editar la expresión.  
  
 **Inferior**  
 Especifique el relleno (o espacio) que hay entre el borde inferior de una celda y el valor mostrado dentro de la celda. El valor predeterminado es 2 puntos. Haga clic en el botón **Expresión** (*fx*) para editar la expresión.
