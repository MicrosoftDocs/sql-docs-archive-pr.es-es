# <a name="uninstall-sql-server-2014"></a>Desinstalar SQL Server 2014
  Consulte los temas siguientes para desinstalar totalmente una instancia existente de [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] y preparar el sistema para poder volver a instalar [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="in-this-section"></a>En esta sección  
 [Desinstalar una instancia existente de SQL Server &#40;programa de instalación&#41;](uninstall-an-existing-instance-of-sql-server-setup.md)  
 En este tema se describe cómo desinstalar manualmente una instancia independiente de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 [Desinstalar PowerPivot para SharePoint](uninstall-power-pivot-for-sharepoint.md)  
 En este tema se describe cómo desinstalar manualmente PowerPivot para SharePoint.  
  
 [Desinstalar Reporting Services](uninstall-reporting-services.md)  
 En este tema se describe cómo desinstalar los servidores de [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] para el modo de SharePoint y los servidores en modo nativo.  
  
 [Desinstalar y quitar Master Data Services](uninstall-and-remove-master-data-services.md)  
 En este tema se describe el proceso para desinstalar y quitar [!INCLUDE[ssMDSshort](../../includes/ssmdsshort-md.md)] del equipo local.  
  
 [Quitar objetos del servidor de calidad de datos](remove-data-quality-server-objects.md)
