# <a name="report-server-properties-advanced-tab"></a>Propiedades de Servidor de informes (pestaña Avanzadas)
  Este es el servicio [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]. Si se han definido propiedades personalizadas, aparecen en esta pestaña junto con los valores correspondientes.  
  
## <a name="options"></a>Opciones  
 **Informes de comentarios de clientes**  
 Indica si se ha habilitado la supervisión de la calidad del servicio en este servicio. Para obtener más información acerca de los informes de comentarios de clientes, busque el tema sobre la configuración de informes de errores y uso en los Libros en pantalla.  
  
 **Volcar directorio**  
 Muestra la ubicación donde se colocan los volcados de memoria en caso de producirse un error.  
  
 **Informes de errores**  
 Cuando se establece en **Sí**, el programa Dr. Watson envía información a [!INCLUDE[msCoName](../../includes/msconame-md.md)] o a su servidor de errores si se produce un error grave. Para obtener más información acerca de Informes de errores, busque el tema sobre la configuración de informes de errores y uso en los Libros en pantalla.  
  
 **Id. de instancia**  
 Indica la instancia que utiliza este servicio.
