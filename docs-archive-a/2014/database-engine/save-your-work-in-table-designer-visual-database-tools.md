# <a name="save-your-work-in-table-designer-visual-database-tools"></a>Guardar el trabajo en el Diseñador de tablas (Visual Database Tools)
  Puede guardar los cambios en las tablas que están abiertas en el Diseñador de tablas.  
  
### <a name="to-save-a-table"></a>Para guardar una tabla  
  
1.  Haga clic en cualquier parte en el Diseñador de tablas.  
  
2.  En el menú **Archivo** , elija **Guardar**_nombreTabla_.  
  
3.  En el cuadro de diálogo **Guardar** , haga clic en **Sí** para actualizar la base de datos.  
  
> [!NOTE]  
>  Si los cambios que se van a guardar no afectan a las demás tablas, no se mostrará este cuadro de diálogo.  
  
> [!NOTE]  
>  La lista de tablas del cuadro de diálogo **Guardar** puede incluir tablas distintas de las que modificó. Por ejemplo, si cambia el tipo de datos de una columna que participa en una relación con otra tabla, se incluirán las dos tablas en esta lista.  
  
> [!NOTE]  
>  Para guardar los cambios realizados en todos los elementos abiertos, como las tablas, vistas y procedimientos almacenados, en el menú **Archivo** , elija **Guardar todos**.
